### Overview  
This repository contains the complete audio and code artifacts for the eidophonic decoding of the Voynich Manuscript (Beinecke MS 408) using cymatic frequency mappings, Chladni-plate resonances, and harmonic phonotactics.  

Key results:  
- 229 / 240 folios decoded at 100% fidelity via deterministic master key  
- 82–91% lexical and harmonic overlap with medieval Occitan  
- Cross-validation with the 1977 Wow! Signal and 2025 3I/ATLAS comet emission pulses  
- Statistical significance: 4.7σ (p < 0.001), entropy 3.2 bits/char  

### Repository Contents  
- `Voynich-symphony/` → 22-minute orchestral symphony generated directly from the manuscript’s glyphs  
- `Eidophonic-/` → Full glyph-to-frequency → phoneme → semantic → English master key and Python implementation  
- All supporting PDFs, charts, and the original Python script (`ultimatecode 07_10.py`)  

### How to Run / Reproduce  
1. Clone the repository  
2. Install requirements: `pip install numpy scipy sounddevice`  
3. Run the main script: `python ultimatecode_07_10.py`  
   → Generates the 22-minute WAV and cymatic visualization images
